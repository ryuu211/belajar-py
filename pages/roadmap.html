Roadmap Belajar Python untuk Pemula

Ikuti roadmap Python ini untuk memahami urutan belajar dari sintaks dasar, struktur data, OOP, praktik file, scraping, sampai spesialisasi seperti data science, web development, dan automation.

1. Dasar-dasar (2-4 Minggu)

  • Instalasi Python & IDE (VS Code)
  • Variabel & Tipe Data (int, float, str, bool)
  • Struktur Kontrol (if, for, while)
  • Fungsi Dasar & Lambda

2. Struktur Data & Modul (2-3 Minggu)

  • List, Tuple, Set, Dictionary
  • Manipulasi String & f-strings
  • Standard Library (math, random, datetime)
  • Pip & Virtual Environments

3. OOP - Object Oriented Programming (2-3 Minggu)

  • Class & Object
  • Inheritance & Polymorphism
  • Encapsulation

4. Intermediate & Praktik (4-6 Minggu)

  • File Handling (CSV, JSON)
  • Error Handling (try-except)
  • Web Scraping (BeautifulSoup)
  • Membuat Mini Project CLI

5. Spesialisasi (Pilih Salah Satu)

  • Data Science: NumPy, Pandas, Matplotlib
  • Web Development: FastAPI, Flask, Django
  • Automation: Selenium, PyAutoGUI
