How-to Guide Python untuk Pemula

Gunakan halaman ini sebagai resep cepat saat Anda ingin menyelesaikan tugas tertentu di Python.

Cara Membuat Fungsi yang Rapi

Gunakan fungsi saat ada kode yang dipakai berulang atau saat satu proses mulai terasa terlalu panjang.

  1. Tentukan satu tujuan fungsi.
  2. Beri nama fungsi dengan kata kerja.
  3. Gunakan parameter untuk data yang berubah.
  4. Gunakan return jika fungsi perlu menghasilkan nilai.
def hitung_total(harga, jumlah):
    return harga * jumlah

total = hitung_total(15000, 3)
print(f"Total belanja: Rp{total}")
Checklist: nama fungsi jelas, tugasnya satu, dan hasilnya mudah diuji.

Cara Membaca dan Menulis File

File handling berguna untuk menyimpan data sederhana tanpa database.

# Menulis file baru
with open("catatan.txt", "w", encoding="utf-8") as file:
    file.write("Belajar Python itu bertahap.\n")

# Menambahkan isi file
with open("catatan.txt", "a", encoding="utf-8") as file:
    file.write("Hari ini belajar file handling.\n")

# Membaca file
with open("catatan.txt", "r", encoding="utf-8") as file:
    isi = file.read()
    print(isi)

Mode file yang sering dipakai

  • "r": membaca file.
  • "w": menulis ulang file dari awal.
  • "a": menambahkan isi di akhir file.

Cara Menangani Error Sederhana

Gunakan try dan except agar program memberi pesan yang jelas saat terjadi masalah.

try:
    angka = int(input("Masukkan angka: "))
    print(100 / angka)
except ValueError:
    print("Input harus berupa angka.")
except ZeroDivisionError:
    print("Angka tidak boleh 0.")

Jangan menangkap semua error tanpa alasan. Tangani error yang memang mungkin terjadi.

Cara Mengambil Data dari API

API memungkinkan program Python mengambil data dari layanan lain. Contoh ini memakai library requests.

# Install dulu:
# pip install requests

import requests

url = "https://api.github.com/users/octocat"
response = requests.get(url, timeout=10)

if response.status_code == 200:
    data = response.json()
    print(data["login"])
    print(data["public_repos"])
else:
    print("Gagal mengambil data API.")

Checklist API

  • Selalu cek status response.
  • Gunakan timeout agar program tidak menunggu terlalu lama.
  • Baca dokumentasi API untuk mengetahui struktur data JSON.
