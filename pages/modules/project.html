10 Project Python Pemula untuk Latihan

Project adalah cara tercepat untuk mengubah teori menjadi kemampuan. Mulai dari project kecil, pahami konsepnya, lalu tambah fitur sedikit demi sedikit.

Kalkulator Tebak Angka To-do List Kasir Catatan File

1. Kalkulator Sederhana

Tujuan: melatih input, konversi tipe data, operator, dan percabangan.

angka_1 = float(input("Angka pertama: "))
operator = input("Operator (+, -, *, /): ")
angka_2 = float(input("Angka kedua: "))

if operator == "+":
    hasil = angka_1 + angka_2
elif operator == "-":
    hasil = angka_1 - angka_2
elif operator == "*":
    hasil = angka_1 * angka_2
elif operator == "/":
    hasil = angka_1 / angka_2
else:
    hasil = "Operator tidak dikenal"

print(f"Hasil: {hasil}")

Ide pengembangan

  • Tangani pembagian dengan 0.
  • Tambahkan operasi pangkat dan modulus.
  • Buat kalkulator berjalan berulang sampai pengguna keluar.

2. Game Tebak Angka

Tujuan: melatih loop, random number, percabangan, dan validasi input.

import random

angka_rahasia = random.randint(1, 10)

while True:
    tebakan = int(input("Tebak angka 1-10: "))

    if tebakan == angka_rahasia:
        print("Benar!")
        break
    elif tebakan < angka_rahasia:
        print("Terlalu kecil.")
    else:
        print("Terlalu besar.")

Ide pengembangan

  • Batasi jumlah percobaan.
  • Simpan skor terbaik.
  • Tambahkan level mudah, sedang, dan sulit.

3. To-do List CLI

Tujuan: melatih list, fungsi, menu terminal, dan pengelolaan data sederhana.

tugas = []

def tambah_tugas(nama):
    tugas.append(nama)

def lihat_tugas():
    for nomor, item in enumerate(tugas, start=1):
        print(f"{nomor}. {item}")

tambah_tugas("Belajar fungsi")
tambah_tugas("Membuat project CLI")
lihat_tugas()

Ide pengembangan

  • Tambahkan fitur hapus tugas.
  • Tambahkan status selesai.
  • Simpan tugas ke file JSON.

4. Program Kasir Sederhana

Tujuan: melatih dictionary, loop, perhitungan total, dan format output.

keranjang = {
    "roti": 12000,
    "susu": 18000,
    "telur": 25000,
}

total = 0

for nama_barang, harga in keranjang.items():
    print(f"{nama_barang}: Rp{harga}")
    total += harga

print(f"Total belanja: Rp{total}")

Ide pengembangan

  • Tambahkan jumlah barang.
  • Hitung diskon.
  • Cetak struk belanja.

5. Aplikasi Catatan Berbasis File

Tujuan: melatih file handling dan mode w, a, serta r.

catatan = input("Tulis catatan: ")

with open("catatan.txt", "a", encoding="utf-8") as file:
    file.write(catatan + "\n")

with open("catatan.txt", "r", encoding="utf-8") as file:
    print(file.read())

Ide pengembangan

  • Tambahkan tanggal otomatis.
  • Buat fitur cari catatan.
  • Tambahkan fitur hapus semua catatan.

6. Kuis Pilihan Ganda

Tujuan: melatih list of dictionary, loop, validasi jawaban, dan skor.

soal = [
    {"tanya": "Tipe data untuk teks?", "jawab": "str"},
    {"tanya": "Keyword membuat fungsi?", "jawab": "def"},
]

skor = 0

for item in soal:
    jawaban = input(item["tanya"] + " ")
    if jawaban.lower() == item["jawab"]:
        skor += 1

print(f"Skor: {skor}/{len(soal)}")

7. Password Generator

Tujuan: melatih string, random, loop, dan penggabungan karakter.

import random
import string

panjang = int(input("Panjang password: "))
karakter = string.ascii_letters + string.digits + "!@#$%"

password = ""
for _ in range(panjang):
    password += random.choice(karakter)

print(password)

8. Cek Profil GitHub dari API

Tujuan: melatih request API, JSON, status code, dan dictionary.

import requests

username = input("Username GitHub: ")
url = f"https://api.github.com/users/{username}"

response = requests.get(url, timeout=10)

if response.status_code == 200:
    data = response.json()
    print(data["login"])
    print(data["public_repos"])
else:
    print("User tidak ditemukan.")

9. Pembaca File CSV

Tujuan: melatih standard library csv dan pengolahan data tabel.

import csv

with open("nilai.csv", "r", encoding="utf-8") as file:
    reader = csv.DictReader(file)
    for baris in reader:
        print(f"{baris['nama']} mendapat nilai {baris['nilai']}")
Buat file nilai.csv dengan kolom nama,nilai sebelum menjalankan contoh ini.

10. Database Kontak dengan SQLite

Tujuan: mengenal database lokal, query SQL dasar, dan penyimpanan permanen.

import sqlite3

conn = sqlite3.connect("kontak.db")
cursor = conn.cursor()

cursor.execute("""
CREATE TABLE IF NOT EXISTS kontak (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nama TEXT,
    nomor TEXT
)
""")

cursor.execute(
    "INSERT INTO kontak (nama, nomor) VALUES (?, ?)",
    ("Ari", "08123456789")
)

conn.commit()
conn.close()

Ide pengembangan

  • Tambahkan fitur lihat semua kontak.
  • Tambahkan fitur ubah nomor.
  • Tambahkan fitur hapus kontak.
