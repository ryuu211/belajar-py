Level 2: Python Menengah

Tingkatkan kemampuan Anda dengan konsep pemrograman yang lebih terstruktur dan efisien.

1. Pemrograman Berorientasi Objek (OOP)

OOP memungkinkan Anda mengorganisir kode menggunakan "Objek" yang memiliki data (atribut) dan perilaku (metode).

class Mobil: def __init__(self, merk, warna): self.merk = merk self.warna = warna def klakson(self): return f"{self.merk} berbunyi: Beep Beep!" # Membuat objek dari class Mobil mobil1 = Mobil("Toyota", "Hitam") print(mobil1.klakson())
Konsep Penting: Inheritance (pewarisan sifat), Polymorphism (banyak bentuk), dan Encapsulation (penyembunyian data).

2. Penanganan File & Error

Belajar mengelola data eksternal dan memastikan program tidak crash saat terjadi kesalahan.

# Membaca File dengan Context Manager try: with open("data.txt", "r") as file: konten = file.read() print(konten) except FileNotFoundError: print("Error: File tidak ditemukan!") finally: print("Operasi selesai.")

3. Fitur Lanjutan (Advanced Features)

Tulis kode yang lebih ringkas (Pythonic) menggunakan Comprehensions dan Decorators.

# List Comprehension (Membuat list baru secara ringkas) angka = [1, 2, 3, 4, 5] kuadrat = [x**2 for x in angka if x % 2 == 0] # Hasil: [4, 16] # Decorator sederhana def cetak_log(fungsi): def wrapper(): print("Log: Fungsi dimulai...") fungsi() print("Log: Fungsi selesai.") return wrapper @cetak_log def halo(): print("Halo Dunia!") halo()
