Modul Python Dasar untuk Pemula

Pelajari fondasi Python dari nol: variabel, tipe data, percabangan, perulangan, koleksi data, fungsi, dan lambda. Modul ini cocok untuk pemula yang baru mulai menulis kode Python.

1. Variabel & Tipe Data

Variabel adalah wadah untuk menyimpan data. Di Python, Anda tidak perlu menentukan tipe data secara manual (Dynamic Typing).

Tipe Data Utama:

  • String (str): Teks, diapit tanda kutip " " atau ' '.
  • Integer (int): Angka bulat tanpa desimal.
  • Float (float): Angka desimal.
  • Boolean (bool): Nilai kebenaran, hanya True atau False.
# Contoh Tipe Data Dasar
nama = "Andi"          # String
umur = 25               # Integer
tinggi = 170.5          # Float
is_active = True        # Boolean

# Konversi Tipe Data (Type Casting)
umur_str = str(umur)    # Mengubah integer ke string
angka_float = float(10) # Hasil: 10.0

print(f"{nama} berumur {umur} tahun.")
Tips: Gunakan nama variabel yang deskriptif (contoh: 'harga_produk' lebih baik daripada 'hp').

2. Struktur Kontrol (Alur Program)

Mengatur jalannya program berdasarkan kondisi tertentu atau pengulangan.

Kondisional (if, elif, else):

nilai = 75
if nilai >= 80:
    print("Nilai A")
elif nilai >= 70:
    print("Nilai B")
else:
    print("Nilai C")

Perulangan (Loops):

# For Loop: Digunakan saat jumlah iterasi sudah diketahui
buah = ["Apel", "Mangga", "Jeruk"]
for b in buah:
    print(f"Buah: {b}")

# While Loop: Digunakan selama kondisi masih True
hitung = 0
while hitung < 3:
    print(f"Hitungan ke-{hitung}")
    hitung += 1

3. Koleksi Data (List, Tuple, Set, Dict)

Python menyediakan struktur data bawaan untuk menyimpan banyak nilai dalam satu variabel.

# List: Urut, bisa diubah (Mutable), boleh duplikat
my_list = [1, 2, 3, "Python"]
my_list.append("Keren") # Menambah data

# Tuple: Urut, TIDAK bisa diubah (Immutable)
my_tuple = (10, 20, 30)

# Set: Tidak urut, unik (tidak boleh duplikat)
my_set = {1, 2, 2, 3} # Hasilnya: {1, 2, 3}

# Dictionary: Pasangan Key-Value
my_dict = {"nama": "Andi", "umur": 25}
print(my_dict["nama"])

4. Fungsi Dasar & Lambda

Fungsi membantu Anda menghindari pengulangan kode (DRY - Don't Repeat Yourself).

# Fungsi Biasa dengan Parameter & Return
def hitung_luas(p, l):
    return p * l

hasil = hitung_luas(10, 5)
print(f"Luas: {hasil}")

# Lambda Function: Fungsi anonim satu baris
kali_dua = lambda x: x * 2
print(f"Hasil: {kali_dua(10)}")
