Level 1: Fondasi Python

Halaman ini membahas segala hal yang perlu Anda ketahui sebagai pemula untuk mulai menulis kode Python.

1. Variabel & Tipe Data

Variabel adalah wadah untuk menyimpan data. Di Python, Anda tidak perlu menentukan tipe data secara manual.

# Contoh Tipe Data Dasar nama = "Andi" # String (teks) umur = 25 # Integer (angka bulat) tinggi = 170.5 # Float (angka desimal) is_active = True # Boolean (True/False) print(f"{nama} berumur {umur} tahun.")
Tips: Gunakan nama variabel yang deskriptif (contoh: 'harga_produk' lebih baik daripada 'hp').

2. Struktur Kontrol (Alur Program)

Gunakan if untuk logika keputusan dan for/while untuk pengulangan.

# Kondisional nilai = 75 if nilai >= 80: print("Nilai A") elif nilai >= 70: print("Nilai B") else: print("Nilai C") # Loop For (Iterasi List) buah = ["Apel", "Mangga", "Jeruk"] for b in buah: print(f"Buah: {b}")

3. Koleksi Data (List, Tuple, Set, Dict)

Python memiliki beberapa cara untuk menyimpan sekumpulan data.

# List: Urut & bisa diubah my_list = [1, 2, 3, "Python"] # Tuple: Urut & TIDAK bisa diubah my_tuple = (10, 20, 30) # Set: Unik & tidak urut my_set = {1, 2, 2, 3} # Hasilnya: {1, 2, 3} # Dictionary: Key-Value pair my_dict = {"nama": "Andi", "umur": 25} print(my_dict["nama"])

4. Fungsi Dasar & Lambda

Fungsi memungkinkan Anda menulis kode sekali dan menggunakannya berkali-kali.

# Fungsi Biasa def hitung_luas(p, l): return p * l print(f"Luas: {hitung_luas(10, 5)}") # Lambda Function (Fungsi satu baris) kali_dua = lambda x: x * 2 print(f"Hasil: {kali_dua(10)}")
