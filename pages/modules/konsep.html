Penjelasan Konsep Python

Halaman ini membantu Anda memahami alasan di balik sintaks Python. Baca bagian ini saat Anda sudah bisa menjalankan contoh kode, tetapi masih ingin tahu cara berpikirnya.

Kenapa Python Cocok untuk Pemula

Python populer karena sintaksnya ringkas dan mirip bahasa manusia. Anda bisa fokus pada logika program sebelum berurusan dengan detail teknis yang lebih rumit.

Sintaks sederhana

Blok kode dibaca lewat indentasi, bukan tanda kurung kurawal.

Ekosistem luas

Python dipakai untuk automation, web, data, AI, scripting, dan testing.

Cepat untuk prototipe

Ide kecil bisa diuji dengan sedikit kode.

Keterbatasannya: Python tidak selalu menjadi pilihan tercepat untuk sistem yang sangat membutuhkan performa rendah-level.

Cara Kerja Variabel dan Tipe Data

Variabel adalah nama yang menunjuk ke sebuah nilai. Python memakai dynamic typing, jadi Anda tidak perlu menulis tipe data saat membuat variabel.

nama = "Budi"       # str
umur = 20           # int
tinggi = 170.5      # float
aktif = True        # bool

Tipe data tetap penting karena menentukan operasi yang bisa dilakukan.

angka = 10
teks = "10"

print(angka + 5)      # valid
print(teks + "5")     # valid, hasilnya "105"
# print(angka + teks) # error: int dan str tidak bisa langsung dijumlahkan

Mutable vs Immutable

Mutable berarti nilai bisa diubah setelah dibuat. Immutable berarti nilai tidak bisa diubah langsung.

Jenis Contoh Makna
Mutable list, dict, set Isi bisa diubah tanpa membuat objek baru.
Immutable str, int, float, tuple Perubahan menghasilkan nilai baru.

OOP dengan Bahasa Sederhana

Object Oriented Programming membantu Anda mengelompokkan data dan perilaku yang saling berhubungan.

  • Class: cetakan atau blueprint.
  • Object: hasil nyata dari class.
  • Atribut: data milik object.
  • Method: fungsi milik object.
class Siswa:
    def __init__(self, nama, nilai):
        self.nama = nama
        self.nilai = nilai

    def lulus(self):
        return self.nilai >= 70

siswa = Siswa("Cici", 85)
print(siswa.nama)
print(siswa.lulus())

Gunakan OOP saat program punya banyak entitas yang memiliki data dan perilaku. Untuk script kecil, fungsi biasa sering lebih sederhana.
