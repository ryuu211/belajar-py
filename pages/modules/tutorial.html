Tutorial Belajar Python dari Nol

Tutorial ini memandu Anda dari program pertama sampai mini project CLI sederhana. Ikuti urutannya, jalankan setiap kode, lalu ubah nilainya agar konsepnya benar-benar terasa.

Program Pertama Variabel dan Input Alur Program Mini Project CLI

1. Menulis Program Python Pertama

Tujuan langkah ini adalah memastikan Python berjalan dan Anda paham cara melihat output program.

  1. Buka editor kode seperti VS Code.
  2. Buat file bernama belajar.py.
  3. Tulis kode berikut, lalu jalankan dengan python belajar.py.
print("Halo, Python!")
print("Saya sedang belajar dari nol.")
Jika output muncul di terminal, berarti lingkungan belajar Anda sudah siap.

2. Menggunakan Variabel dan Input

Variabel menyimpan nilai agar bisa dipakai lagi. Input membuat program menerima data dari pengguna.

nama = input("Masukkan nama: ")
umur = int(input("Masukkan umur: "))

print(f"Halo {nama}, tahun depan umurmu {umur + 1} tahun.")

Yang perlu diperhatikan

  • input() selalu menghasilkan string.
  • Gunakan int() jika input harus menjadi angka bulat.
  • Gunakan nama variabel yang jelas seperti umur, bukan x.

3. Membuat Keputusan dan Perulangan

Program menjadi berguna ketika bisa mengambil keputusan dan mengulang pekerjaan.

nilai = int(input("Masukkan nilai: "))

if nilai >= 80:
    print("Bagus, lanjutkan latihan proyek.")
elif nilai >= 60:
    print("Cukup, ulangi materi dasar.")
else:
    print("Pelajari lagi contoh kode dan latihan kecil.")

for nomor in range(1, 4):
    print(f"Latihan ke-{nomor}")

Gunakan if untuk percabangan. Gunakan for saat jumlah pengulangan sudah diketahui.

4. Mini Project: Daftar Tugas CLI

Project ini melatih list, loop, fungsi, dan input. Program menyimpan daftar tugas selama program berjalan.

tugas = []

def tampilkan_menu():
    print("\n=== Daftar Tugas ===")
    print("1. Tambah tugas")
    print("2. Lihat tugas")
    print("3. Hapus tugas")
    print("4. Keluar")

def tambah_tugas():
    nama_tugas = input("Tugas baru: ")
    tugas.append(nama_tugas)
    print("Tugas ditambahkan.")

def lihat_tugas():
    if not tugas:
        print("Belum ada tugas.")
        return

    for index, item in enumerate(tugas, start=1):
        print(f"{index}. {item}")

def hapus_tugas():
    lihat_tugas()
    if not tugas:
        return

    nomor = int(input("Nomor tugas yang dihapus: "))
    if 1 <= nomor <= len(tugas):
        tugas.pop(nomor - 1)
        print("Tugas dihapus.")
    else:
        print("Nomor tidak valid.")

while True:
    tampilkan_menu()
    pilihan = input("Pilih menu: ")

    if pilihan == "1":
        tambah_tugas()
    elif pilihan == "2":
        lihat_tugas()
    elif pilihan == "3":
        hapus_tugas()
    elif pilihan == "4":
        print("Sampai jumpa!")
        break
    else:
        print("Pilihan tidak tersedia.")

Tantangan lanjutan

  • Tambahkan status selesai atau belum selesai.
  • Simpan data ke file JSON.
  • Tambahkan pencarian tugas berdasarkan kata kunci.
