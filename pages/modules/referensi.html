Tipe Data Utama

Tipe Fungsi Contoh
str Menyimpan teks. "Python"
int Menyimpan angka bulat. 10
float Menyimpan angka desimal. 3.14
bool Menyimpan nilai benar atau salah. True, False
list Menyimpan banyak data yang bisa diubah. [1, 2, 3]
dict Menyimpan pasangan key-value. {"nama": "Ari"}

Standard Library yang Sering Dipakai

Library Kegunaan Contoh Pemakaian
math Operasi matematika. math.sqrt(16)
random Menghasilkan pilihan acak. random.choice(data)
datetime Mengelola tanggal dan waktu. datetime.now()
json Membaca dan menulis data JSON. json.load(file)
csv Membaca file CSV. csv.DictReader(file)
sqlite3 Menggunakan database SQLite. sqlite3.connect("app.db")

Error Python Umum

Error Penyebab Umum Cara Memperbaiki
SyntaxError Sintaks tidak valid. Cek tanda titik dua, kurung, dan indentasi.
NameError Variabel belum dibuat atau salah ketik. Pastikan nama variabel sama persis.
TypeError Operasi dilakukan pada tipe data yang tidak cocok. Konversi tipe data jika perlu.
ValueError Nilai tidak sesuai dengan tipe yang diminta. Validasi input sebelum dikonversi.
KeyError Key dictionary tidak ditemukan. Gunakan dict.get() atau cek key terlebih dahulu.
FileNotFoundError File yang dibuka tidak ada. Periksa path file dan nama file.
