Modul Python Lanjutan: Scraping, Data, dan SQLite

Terapkan Python untuk kebutuhan praktis seperti web scraping, integrasi API, analisis data dengan NumPy dan Pandas, database SQLite, dan otomatisasi sederhana.

1. Web Scraping & API

Mengambil data secara otomatis dari website menggunakan library requests dan BeautifulSoup.

import requests
from bs4 import BeautifulSoup

url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# Mengambil semua tag h1 di halaman
for h1 in soup.find_all('h1'):
    print(h1.text)
Kebutuhan: Install library dengan pip install requests beautifulsoup4.

2. Analisis Data (NumPy & Pandas)

Python adalah raja dalam data science. Gunakan Pandas untuk mengelola data tabel (DataFrame).

import pandas as pd
import numpy as np

# Membuat dataset sederhana
data = {
    'Nama': ['Andi', 'Budi', 'Cici'],
    'Skor': [85, 90, 78]
}
df = pd.DataFrame(data)

# Menghitung rata-rata skor
rata_rata = df['Skor'].mean()
print(f"Rata-rata Skor: {rata_rata}")

3. Database (SQLite)

Menyimpan data secara permanen menggunakan SQL database yang terintegrasi dengan Python.

import sqlite3

# Koneksi ke database (akan dibuat jika belum ada)
conn = sqlite3.connect('belajar.db')
cursor = conn.cursor()

# Membuat tabel
cursor.execute('CREATE TABLE IF NOT EXISTS user (nama TEXT, umur INTEGER)')

# Menambah data
cursor.execute('INSERT INTO user VALUES ("Dedi", 22)')
conn.commit()
conn.close()
